### 2. 77GHz 车载 4D 成像毫米波雷达算法开发与研制

我曾从事 **77GHz 车载 4D 成像毫米波雷达** 的算法开发和研制，关注从底层信号处理到车载应用输出的完整流程。

涉及方向包括：

- 77GHz FMCW 雷达信号处理；
- Range FFT / Doppler FFT / Angle FFT；
- 距离、速度、角度估计；
- 4D 点云生成；
- 多目标检测；
- CFAR 检测；
- 点云聚类与目标跟踪；
- 车载场景下的目标稳定性分析；
- 阵列通道幅相一致性；
- DOA 估计与角度误差分析；
- 前向雷达、BSD、泊车、ACC 等应用场景理解。

在车载雷达方向，我关注的不只是“检测到目标”，更关注：

- 目标是否稳定；
- 点云是否连续；
- 速度方向是否正确；
- 角度是否可信；
- 虚警和漏检来自哪里；
- 近距离、远距离、大角度、低速目标、静态目标分别应该如何处理；
- 算法输出如何服务于真实车载功能。

我尝试使用 **24GHz 和 60GHz 雷达** 实现多种智能家居与康养场景应用，包括：

- 非接触式呼吸检测；
- 非接触式心率检测；
- 人体存在检测；
- 人体定位；
- 手势识别；
- 跌倒检测；
- 室内人体检测；
- 多目标生命体征检测；
- 机器人搭载雷达感知；
- 隔门移动扫地机器人搭载雷达实现人体姿态感知与呼吸心率检测；
- 电风扇场景下人体定位感知。

其中，**60GHz 呼吸心率检测雷达已经实现量产交付**。围绕这一方向，我对以下问题做过持续研究：

- 胸腔微动如何映射到雷达相位；
- 呼吸信号和心跳信号如何分离；
- 静态杂波如何去除；
- 体动如何影响生命体征；
- 多通道相位如何融合；
- 心率为什么比呼吸难；
- 多人目标在同距离门下如何分离；
- 雷达心率如何与指夹仪、心电仪进行对比；
- 工程产品中如何处理跳变、丢帧、低信噪比和误检。

### 4. UWB 雷达近端感知研究

后续我开始尝试使用 **UWB 雷达**，并系统学习了 UWB 雷达的波形、帧结构、CIR、相关接收、脉冲压缩和近端感知原理。

关注方向包括：

- UWB 脉冲雷达基本原理；
- IEEE 802.15.4z 相关机制；
- SYNC 字段与雷达感知；
- CIR 数据结构；
- 快时间 / 慢时间处理；
- 近端目标检测；
- 静态杂波抑制；
- 人体存在检测；
- 呼吸微动检测；
- 车内儿童遗留检测；
- 近距离障碍物感知；
- UWB 与毫米波雷达、超声波雷达的能力边界对比。

我对 UWB 雷达的理解不是停留在“它能测距很准”，而是更关注：

- UWB 的大带宽如何形成距离分辨能力；
- CIR 中的峰值、相位和多径如何理解；
- UWB 在近端感知中的优势和盲区；
- UWB 是否适合替代超声波雷达；
- UWB 与车载数字钥匙、车内感知、近端雷达化应用如何结合；
- UWB 感知应用真正落地时会遇到哪些工程限制。

## 我对雷达的理解

雷达不是一个单独的算法，也不是一个单独的芯片。

一个真正能落地的雷达系统，至少要同时理解：

- 波形怎么发；
- 天线怎么收；
- 阵列怎么排；
- 回波怎么形成；
- 数据怎么采；
- FFT 怎么做；
- 杂波怎么抑制；
- 目标怎么检测；
- 角度怎么估计；
- 点云怎么生成；
- 虚警怎么压；
- 漏检怎么补；
- 跟踪怎么稳定；
- 场景怎么约束；
- 产品怎么定义；
- 用户真正需要什么。

我希望持续把这些内容沉淀下来，形成一个面向工程实践的雷达技术知识库。
